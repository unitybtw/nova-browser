# Open Source Browser Architecture & Specification

## Core Features
1. **Vertical Tabs & Horizontal Tabs**: Dynamic switching with framer-motion smooth animations.
2. **Workspace Isolation**: Manage distinct contexts with individual tab filtering and accent colors.
3. **AdBlocker & Privacy Shield**: Integrated `@cliqz/adblocker-electron` for fast content filtering.
4. **AI Assistant Side Panel**: Context-aware side panel interface with TTS integration.
5. **Split View Screen**: Run two active tabs side-by-side.
